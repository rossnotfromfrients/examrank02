## Subject

```c
Nombre de la tarea: ft_rrange
Archivos esperados: ft_rrange.c
Funciones permitidas: malloc
--------------------------------------------------------------------------------

Escribe la siguiente función:

int     *ft_rrange(int start, int end);

Debe asignar memoria (con malloc()) para un array de enteros, llenarlo con valores consecutivos
que comiencen en end y terminen en start (¡Incluyendo start y end!), luego
devolver un puntero al primer valor del array.

Ejemplos:

- Con (1, 3) devolverás un array que contenga 3, 2 y 1.
- Con (-1, 2) devolverás un array que contenga 2, 1, 0 y -1.
- Con (0, 0) devolverás un array que contenga 0.
- Con (0, -3) devolverás un array que contenga -3, -2, -1 y 0.

```
